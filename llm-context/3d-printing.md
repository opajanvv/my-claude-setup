# 3D printing

last reviewed: 2026-02-19

## Setup

- **Printer:** Bambu Lab A1 (bedslinger, full size)
- **Color:** Single color — no AMS
- **Filament:** PLA (standard)
- **Slicer:** OrcaSlicer

## Design tools

Both tools are in active use depending on the task:

- **TinkerCad** — web-based, GUI-driven parametric design; good for quick models and beginner-friendly shapes
- **OpenSCAD** — code-based parametric design; good for precise, reusable, or mathematically defined models

## Model sources

- **Printables.com** — primary source for downloading models
- **Makerworld** — secondary source

## Typical workflows

**Download path:** find model on Printables → open STL in OrcaSlicer → slice with A1 profile → send to printer

**Design path:** model in TinkerCad or OpenSCAD → export STL → open in OrcaSlicer → slice → print
